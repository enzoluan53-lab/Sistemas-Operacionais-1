# 📘 Aula 03 – Conceitos, Funções e Tipos de Sistemas Operacionais

## 🧠 Introdução aos Sistemas Operacionais

Sistemas Operacionais (SO) são responsáveis por gerenciar recursos do computador e fornecer interface entre usuário e hardware.

---

## 🖥️ Tipos de Sistemas Operacionais

### 🏢 Sistemas de Grande Porte (Mainframes)

* Alto desempenho para grandes volumes de dados
* Suporte a muitos usuários simultâneos
* Alta confiabilidade e segurança
* Uso: bancos, e-commerce, grandes empresas
* Exemplos: OS/360, OS/390, Linux, UNIX

---

### 🌐 Sistemas Operacionais de Servidor

* Atendem múltiplos usuários via rede
* Foco em estabilidade e escalabilidade
* Serviços: web, banco de dados, arquivos
* Exemplos:

  * Linux
  * Windows Server

---

### ⚙️ Sistemas Multiprocessadores

* Utilizam múltiplas CPUs ou núcleos
* Permitem processamento paralelo
* Desafios:

  * Escalonamento de tarefas
  * Sincronização (locks, semáforos)
  * Coerência de cache
* Uso: servidores e computação científica

---

### 💻 Sistemas de Computadores Pessoais

* Voltados para uso individual
* Interface gráfica e suporte multimídia
* Exemplos:

  * Windows
  * macOS
  * Linux

---

### 📱 Sistemas Operacionais Portáteis

* Uso em smartphones e tablets
* Foco em:

  * Economia de energia
  * Segurança (permissões)
  * Integração com sensores
* Exemplos:

  * Android
  * iOS

---

### 🔌 Sistemas Embarcados

* Executados em dispositivos dedicados
* Recursos limitados (memória, processamento)
* Aplicações:

  * Eletrodomésticos
  * Automóveis
  * Smart TVs
* Exemplos: Embedded Linux, QNX, VxWorks

---

### 📡 Sistemas de Nós Sensores

* Dispositivos pequenos com bateria limitada
* Comunicação sem fio
* Uso:

  * Monitoramento ambiental
  * Agricultura
* Exemplos: TinyOS, Contiki

---

### ⏱️ Sistemas de Tempo Real

* **Hard Real-Time**: falhas são críticas (ex: aviões)
* **Soft Real-Time**: pequenas falhas são toleráveis (ex: streaming)

---

### 💳 Sistemas de Cartões Inteligentes

* Recursos muito limitados
* Foco em segurança (criptografia, autenticação)
* Execução de pequenos programas (applets)

---

## 🔧 Controle de Versão com Git

### 📌 O que é Git?

* Sistema de controle de versão
* Permite rastrear mudanças em arquivos
* Integração com plataformas como GitHub

---

### ⚙️ Principais Funções

* Versionamento de código
* Backup e restauração de versões
* Colaboração em projetos

---

### 💻 Instalação do Git

1. Baixar no site oficial
2. Instalar (Next > Next > Install)
3. Testar com:

   ```bash
   git --version
   ```

---

### 🔑 Configuração Inicial

```bash
git config --global user.name "Nome"
git config --global user.email "Email"
```

---

### 📁 Criando Repositório (VS Code)

* Criar pasta e abrir no VS Code
* Criar arquivo (.md)
* Inicializar repositório
* Fazer commit

---

### ☁️ Publicando no GitHub

* Publicar branch
* Escolher repositório (público/privado)
* Acessar pelo GitHub

---

## ✅ Boas Práticas com Git

* Commits pequenos e frequentes
* Mensagens claras
* Uso de branches
* Testes antes de integrar (merge)
